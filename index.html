# Notifications

  1. This website also offers a Chinese version (opens new window). In case of any discrepancies between the English and Chinese versions, the English version will take precedence.

  3. Regarding App Development, you need to understand the definition and structure of the aiM18 App. We suggest that you read the content of app development in sequence.
  4. Regarding API Integration, you need to first understand how aiM18 APIs are authorized, and then you can search for API information based on keywords.
